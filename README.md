# MCFruit Website

A modern, multi-page website for the MCFruit Minecraft 1.21+ Practice Network.

## Pages

- **index.html** - Homepage with hero section, features, and server status
- **rules.html** - Server rules and punishment guidelines
- **staff.html** - Staff team showcase with roles and application info
- **about.html** - About MCFruit, technology, and community info

## Setup

1. Update `script.js` to point to your server's API endpoint (if needed)
2. Customize staff members in `staff.html`
3. Edit rules in `rules.html` as needed
4. Deploy to your web server

## Server Status

The website automatically checks the Minecraft server status using the mcsrvstat.us API. It displays:
- Online/Offline status
- Current player count
- Server IP (mcfruit.club)
